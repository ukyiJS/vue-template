# vue-template

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production build

```
yarn build
```

### require ENV

```
VUE_APP_CRYPTO_SECRET_KEY
```
